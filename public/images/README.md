# Pasta de Imagens e Assets do Site - Dedetizadora Maringá

Coloque aqui todas as imagens originais da marca e fotos do site:
- `logo.png` (Logo da Maringá Dedetizadora e Desentupidora)
- `hero-bg.jpg` ou `tecnico.png` (Fotos dos técnicos, viaturas ou equipamentos)
- Outras imagens e ícones

## Como Referenciar no Código
Qualquer imagem colocada nesta pasta (`public/images/`) pode ser referenciada diretamente no código React/HTML usando a barra inicial:
```tsx
<img src="/images/logo.png" alt="Dedetizadora Maringá" />
```
